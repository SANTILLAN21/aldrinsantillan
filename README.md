# Aldrin B. Santillan – Portfolio

## Setup
1. Extract the zip into any folder
2. Add your assets:
   - `images/profile.jpg`   → Your professional photo
   - `images/logo.png`      → Your personal logo
   - `images/favicon.png`   → Your favicon
   - `resume/Aldrin_Santillan_CV.pdf` → Your CV
3. Open `index.html` in a browser — no build step needed

## Structure
```
portfolio/
├── index.html
├── css/style.css
├── js/script.js
├── images/
│   ├── profile.jpg
│   ├── logo.png
│   └── favicon.png
└── resume/
    └── Aldrin_Santillan_CV.pdf
```
